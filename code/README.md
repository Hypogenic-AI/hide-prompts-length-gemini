# Cloned Repositories

## 1. LLM Attacks (GCG)
- **URL**: https://github.com/llm-attacks/llm-attacks
- **Purpose**: Official implementation of the "Universal and Transferable Adversarial Attacks" paper. Contains the GCG attack algorithm and the `AdvBench` dataset.
- **Location**: `code/llm-attacks/`

## 2. Cognitive Overload Attack (1)
- **URL**: https://github.com/UNHSAILLab/working-memory-attack-on-llms
- **Purpose**: Implementation of "Cognitive Overload Attack: Prompt Injection for Long Context". Contains methods to overload working memory.
- **Location**: `code/cognitive-overload-1/`

## 3. Cognitive Overload (2)
- **URL**: https://github.com/luka-group/CognitiveOverload
- **Purpose**: Implementation of "Cognitive Overload: Jailbreaking Large Language Models with Overloaded Logical Thinking". Alternative cognitive overload approach.
- **Location**: `code/cognitive-overload-2/`

## 4. Lost in the Middle
- **URL**: https://github.com/nelson-liu/lost-in-the-middle
- **Purpose**: Official code for "Lost in the Middle: How Language Models Use Long Contexts". Contains evaluation scripts for multi-document QA and key-value retrieval tasks.
- **Location**: `code/lost-in-the-middle/`
